# setup

### 1.) clone repo
### 2.) install tmux 
```
... install tmux
```
### 3.) copy tmux file from repo to ~/.tmux.conf and source
```
tmux
tmux source-file ~/.tmux.conf
```
### 4.) install neovim
```
# on linux install from PPA
sudo add-apt-repository ppa:neovim-ppa/unstable -y
sudo apt update
sudo apt install neovim -y
```
